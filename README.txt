이 프로그램은 스도쿠퍼즐을 입력받아 그 답을 구하는 프로그램입니다.
프로그램은 inputf.py process.py printf.py project.py 4가지로 구성되어있습니다.

프로그램은 다음과 같은 순서로 작동됩니다.
1.스도쿠 퍼즐을 입력받습니다. 이때 사용자는 스도쿠 퍼즐을 프로그램 안에서 직접 입력할 수 있고 텍스트 파일로도 입력할 수 있습니다.
1-1.빈칸은 0으로 입력합니다.
1-2.9개의 숫자를 입력하지 않거나 중복되는 숫자가 있을 경우 오류메시지를 출력합니다.

2.입력받은 데이터를 가지고 각각의 행, 열, 사각형에 중복된 숫자들의 집합을 구합니다.
2-1.먼저 입력받은 데이터는 2차원 리스트의 형태로 저장이 됩니다.
ex) List[3][5] 이면 4행 6열의 숫자를 의미
2-2.입력받은 데이터로부터 열, 사각형에 대한 정보도 2차원 리스트로 만듭니다.
ex)ListofRow[4][7]이면 5열의 8번째 숫자를 의미
ex)ListofRec[1][2]이면 2번째 사각형의 3번째 숫자를 의미
2-3.3개의 2차원 리스트로부터 중복된 숫자를 가지는 집합을 원소로하는 리스트를 만듭니다.
ex)ListofColset[5]은 6행에 현재 존재하는 숫자들을 원소로 가지는 집합{2, 4, 7}
ex)ListofRowset[2]은 3열에 현재 존재하는 숫자들을 원소로 가지는 집합{1, 9}
ex)ListofRecset[8]은 9번째 사각형에 현재 존재하는 숫자들을 원소로 가지는 집합{5,6}

3.구한 정보를 가지고 빈칸들을 조사하여 빈칸에 들어갈 수 있는 숫자를 구합니다.
3-1.집합{1,2,3,4,5,6,7,8,9}에서 빈칸에 해당하는 행, 열, 사각형에 중복되는 숫자를 뺍니다.
3-2.만약 빈칸에 들어갈 수 있는 숫자가 하나면 답으로 확정.
3-3.만약 빈칸에 들어갈 수 있는 숫자가 여럿이면 가능한 숫자들의 목록만 보여준다.

4.확정된 답과 빈칸의 들어갈 수 있는 숫자들을 출력한다.
4-1.만약 사용자가 답을 더 구하고 싶을 경우 위의 순서를 다시 반복.
4-2.빈칸을 모두 숫자로 채웠으면 프로그램을 종료합니다.
